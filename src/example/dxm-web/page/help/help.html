怎么下载并注册贷小盟APP

扫描二维码下载APP，即可使用手机号注册。

怎么通过贷小盟注册/绑定商户

可进入贷小盟商户推荐界面，点击商户详情页，点击“立即申请”进入下载界面，使用号码注册商户，进行信息认证即可。

下载/注册商户显示“操作失败”

您可先检查网络连接状况是否良好，在保持网络通畅的情况下再操作试一试。如仍未解决问题，请联系贷小盟官方客服。

平台定位

提供需要用钱的老百姓和有钱资本方之间的平台。

平台目的

把线下银行搬到手机上来，让每个会员都成为自己的掌上银行家。

平台为什么要跟我们合作？

市场初期，平台需要我们协助平台建立大数据。

会员如何贷到款？

根据各商户资信要求提供审核资料，具体是否放款以商户的信审结果为准。

会员借钱需要抵押担保吗？

不需要。

会员借钱有黑白户要求吗？

没有。

不会，因为跑了受损失的是公司和平台，平台反而担心的是会员跑路。

公司是直销模式吗？

不是，属分享经济，也叫众享经济，大众创业。

公司平台是一种什么性质？

互联网金融创新模式，万众创新。
